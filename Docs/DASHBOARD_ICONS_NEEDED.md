## Icon Specifications:

- **Format**: PNG with transparent background
- **Size**: 64x64px to 128x128px (will be displayed at 48x48px)
- **Style**: Simple, flat design matching real dashboard icons
- **Color**: Yellow/Amber for most warnings (colors can be adjusted in CSS)
- **Background**: Transparent

## Important Notes:

1. Make sure icons match ACTUAL dashboard warning lights from real vehicles
2. Use consistent style across all icons
3. Ensure icons are recognizable at small sizes (48x48px)
4. Keep file names exactly as listed above for the code to work properly

## After Downloading:

1. Save all icons to: `f:\CBD Pannel\CBD Panelbeating Mechanical\public\Dashboard Icons\`
2. Verify file names match exactly (case-sensitive on some systems)
3. Test the webpage to ensure all icons load correctly
4. The code will automatically use the new icons once they're in place
